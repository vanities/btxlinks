What is BTX?

BTX is a post-quantum hard fork of Bitcoin Knots. It replaces traditional hashing proof-of-work with matrix multiplication — work that maps cleanly onto GPUs and CPUs — and swaps ECDSA signatures for quantum-resistant, lattice-based cryptography.

This is an open-source, community-maintained directory. The source lives on GitHub — to add or fix a link, edit index.html and open a pull request.
